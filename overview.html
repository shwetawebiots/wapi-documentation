Overview

Getting Started

1. Unzip Downloaded Project:

- Unzip the file which you got from CodeCanyon and to see the directories.

2. Explore Wapi Files:

- After extracting the file, you'll see that Wapi is a three primary parts: wapi-api, wapi-frontend, and wapi-admin. You need to run each part separately and in a specific order.

Wapi Package :

  • 1. Wapi-api
  • 2. Wapi-frontend
  • 3. Wapi-admin
  • 4. Wapi-documentation

To run the application, make sure you install all three parts of wapi. Following this simple step-by-step installation guide.
