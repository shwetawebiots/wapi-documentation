Developer API

Overview

The WAPI platform provides a comprehensive REST API that enables developers to integrate WhatsApp marketing capabilities into their own applications and services. Our API follows industry-standard practices and provides secure, reliable access to all platform features.

API Authentication

All API requests require authentication using API keys. Each user account receives a unique API key that must be included in the Authorization header of each request.

  • API Key Generation: Navigate to Settings → API Keys to generate your API credentials
  • Authentication Header: Include the header "Authorization: Bearer YOUR_API_KEY" with each request
  • Security: Store your API keys securely and never expose them in client-side code

Available Endpoints

Our API provides access to all major platform features:

  • Message Management: Send, receive, and manage WhatsApp messages programmatically
  • Template Operations: Create, update, and manage message templates
  • Contact Management: Add, update, and organize contacts and customer profiles
