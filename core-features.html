Wapi - Core Features

Essential tools for WhatsApp marketing automation and business growth.

Admin Dashboard & Analytics

Comprehensive overview of business metrics and performance.

  • Revenue Analytics:
    Track daily, monthly, and total revenue with visual charts.
  • User Statistics:
    Monitor total users, active subscriptions, and user growth.
  • Plan Performance:
    View revenue breakdown by subscription plans.
  • Subscription Management:
    Track new, active, and cancelled subscriptions in real-time.
  • Contact Inquiries:
    Manage customer inquiries and support requests.

User Management System

Complete control over customer accounts and subscriptions.

  • User Database:
    Manage all customer profiles, contact information, and preferences.
  • Subscription Tracking:
    Monitor user subscription status, plans, and billing cycles.
  • Bulk Operations:
    Perform mass actions like status updates or plan changes.
  • User Filtering:
    Search and filter users by various criteria.
  • Profile Management:
    View and edit user details, plans, and account status.

Subscription Plan Management

Create and manage flexible pricing plans for your customers.

  • Plan Creation:
    Design custom subscription plans with different features and pricing.
  • Billing Cycles:
    Support monthly, yearly, and custom billing periods.
  • Feature Limits:
    Set usage limits and feature access per plan tier.
  • Plan Analytics:
    Track plan performance and revenue generation.
  • Bulk Plan Management:
    Update multiple plans simultaneously.

AI Model Integration

Powerful AI capabilities for automated customer interactions.

  • Multiple AI Providers:
    Integrate with OpenAI, Anthropic, Google, and custom AI models.
  • Model Configuration:
    Set up API endpoints, keys, and model parameters.
  • Default Model Selection:
    Choose which AI model to use as the default for chatbots.
  • Status Management:
    Enable/disable AI models based on performance or maintenance.
  • Performance Monitoring:
    Track AI model usage and response quality.

Template Management System

Create and manage pre-built message templates for consistent branding.

  • Template Creation:
    Design professional message templates with variables and formatting.
  • Category Organization:
    Organize templates by sector (business, education, healthcare, etc.).
  • Approval Workflow:
    Manage template approval status for quality control.
  • Multi-language Support:
    Create templates in different languages for global reach.

Payment Gateway Integration

Secure payment processing for subscription management.

  • Stripe Integration:
    Accept credit/debit card payments with Stripe's secure platform.
  • Razorpay Integration:
    Process payments through India's leading payment gateway.
  • Subscription Billing:
    Automated recurring payments and subscription management.
  • Transaction Tracking:
    Monitor all payment transactions and revenue flow.
  • Refund Management:
    Process customer refunds and handle payment disputes.

Landing Page Builder

Create professional marketing pages to attract customers.

  • Drag & Drop Editor:
    Intuitive interface for building custom landing pages.
  • Hero Section:
    Create compelling headlines and call-to-action sections.
  • Feature Showcase:
    Highlight key platform features and benefits.
  • Pricing Tables:
    Display subscription plans with clear pricing information.
  • Testimonial Integration:
    Showcase customer reviews and success stories.

Content Management

Manage FAQs, testimonials, and support content efficiently.

  • FAQ Management:
    Create and organize frequently asked questions with categories.
  • Testimonial System:
    Collect and display customer testimonials with ratings.
  • Content Filtering:
    Easily search and filter content by status or category.
  • Bulk Operations:
    Perform mass updates, deletions, or status changes.
  • Moderation Tools:
    Review and approve user-generated content before publishing.
